# Gazprom Corporate Surveys API

This is a simple API for Gazprom Corporate Surveys application.
It is written in C#/.NET 8 and uses MS SQL Server as a database.

## How to run

### Prerequisites

- .NET 8 SDK
- MS SQL Server 2019 (or Docker)

### Running

```bash
docker-compose up -d
dotnet run
```
